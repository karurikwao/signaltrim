# Changelog

All notable SignalTrim changes are tracked here.

## [0.6.4] - 2026-07-05

### Added

- Launch-ready README, Wiki source, Cloudflare Pages launch site, and generated demo assets.
- Cross-agent installer surface for Claude Code, Codex, Gemini, Cursor, Windsurf, opencode, OpenClaw, Hermes, and npx-skills profiles.
- Local proof features: `/signaltrim-stats`, statusline savings suffix, MCP shrink proxy, and safer memory compression workflow.
- Platform release bundles for Windows, macOS, and Linux generated by the GitHub release workflow.

### Changed

- Release automation now verifies the tag matches `package.json`, runs the standalone JS regression suite, smokes the generated package artifact, and publishes only file assets.
- Public package badge now describes the GitHub package version instead of implying npm registry publication.
- Install and security docs now state the actual detached-download trust model: selected repository ref plus SHA-256 manifest verification.

### Fixed

- CI now runs the root-level standalone JS regression tests that previously lived outside npm scripts.
- Release artifact upload no longer includes staging directories.
