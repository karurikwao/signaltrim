# Examples

Use these pages when you already know which SignalTrim surface you are wiring and need copyable patterns.

## Claude Code Hooks

[Claude Code hook templates](https://github.com/karurikwao/signaltrim/blob/main/docs/examples/claude-code-hooks.md)

Includes standalone hook install shapes, mode tracking examples, statusline notes, and safety rules for preserving user-authored hooks.

Related wiki page: [Claude Code Hooks](Claude-Code-Hooks.md).

## MCP Shrink

[MCP Shrink presets](https://github.com/karurikwao/signaltrim/blob/main/docs/examples/mcp-shrink-presets.md)

Includes filesystem, GitHub, memory/search, and browser-style upstream examples. Bare `--with-mcp-shrink` is invalid because the proxy needs an upstream MCP command.

Related wiki page: [MCP Shrink](MCP-Shrink.md).
