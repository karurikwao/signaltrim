Show the signaltrim quick-reference card.

| Command | What |
|---|---|
| `/signaltrim` | Activate at default level (full) |
| `/signaltrim lite` | Light compression — ~30% tokens dropped |
| `/signaltrim ultra` | Maximum compression |
| `/signaltrim wenyan[-lite\|-ultra]` | Classical Chinese compression |
| `/signaltrim off` | Deactivate |
| `/signaltrim-commit` | Terse commit message |
| `/signaltrim-review` | One-line review findings |
| `/signaltrim-compress <file>` | Compress a Markdown file |
| `/signaltrim-stats` | Lifetime token-savings |

Natural language also works: "turn on signaltrim", "stop signaltrim", "normal mode".

Code, commits, security warnings: signaltrim drops out automatically.
